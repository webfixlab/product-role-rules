== Description ==

EASY to use and super FAST WooCommerce product role based pricing solution to add different prices for different roles. Great for accessories, restaurant or any WooCommerce shops, great for increase conversions.

Role Based Pricing is a truly lightweight and simple yet powerful plugin without any unnecessary features, no huss fuss or no low page speed score!

**FEATURES**

* New! WooCommerce High-Performance Order Storage (HPOS) enabled
* All product types supported
* Dynamic pricing for different user roles and non-registered visitors
* Disable role based pricing temporarily
* Hide price and add placeholder text
* Add new roles

== Installation ==

1. Upload the plugin folder "product-role-rules" to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to options page in the left menu bar of the dashboard **Role Pricing**

== Frequently Asked Questions ==

= How can I set the price for different roles? =

Go to options page in the left menu bar of the dashboard **Role Pricing** and set your prices for each user roles.

= Will it work on any theme or page builder? =

Yes, it will work fine on any standard theme or page builder like Gutenberg, Elementor, DIVI, WPBakery Composer etc.
